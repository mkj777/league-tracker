## HUBRIS.GG

HUBRIS.GG is a League of Legends Profile Tracker aiming to be less cluttered, less analytical and more precise about what you really want to see to climb.  
The Website is trying to be a Hub for League of Legends Players that are trying to climb.  
More focused on a thriving Community than an analytical AI sloppy data powerhouse.

### Deployment

as its currently in Development the Website will be hosted on hubris.gg, which currently only has a placeholder active.

### Tech Stack

Frontend: Vite React Typescript with Tailwind and shadcn UI Components  
Backend: Javascript Express Backend using Axios and a PostgreSQL Database
